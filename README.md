# ⚡ Local Waste & Recycling Incentive System

A full-stack Streamlit web application that encourages proper waste segregation and recycling by awarding points to users and collectors based on their submissions.

## 🚀 Project Overview

This application allows users to submit wet or dry waste and earn points for proper segregation. Collectors are assigned specific areas and also earn points for collecting waste. The system features leaderboards, warnings for improper submissions, and the ability to generate train/test CSVs for analysis.

## 📊 Input Features

Users submit the following information through the web interface:

- 👤 Name  
- 🏠 Area  
- ♻ Waste Type (Plastic, Paper, Organic, Other)  
- ⚖ Quantity (kg)  
- ✅ Proper/Improper Segregation  

## 🎯 Output

- 🌱 Points earned by user  
- 🗂 Waste category (Wet/Dry)  
- 📈 Updated leaderboards for users and collectors  
- ⚠️ Warnings for improper submissions  

## 🛠 Technologies Used

- Python  
- Streamlit  
- Pandas  
- Scikit-learn  
- NumPy
